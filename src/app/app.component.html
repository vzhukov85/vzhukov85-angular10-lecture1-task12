Первая лекция по angular

На петвой лекции было мало новой информации. Из интересного:
  • Знакомство с Angular CLI
  • общая структура проекта, компиляция проекта

Рекомендации:
  • процесс разверывания окружения можно вынести за рамки курса как предварительное задание: trello, visual code, учетка на git/github
  • большинство разработчиков знакома с базовым инструментарием git и github, для тех кто не знаком, можно вынести в пререквизиты для начала работы
  • возможно следует дать ссылку на минимальный ликбез, для тех, кто не знаком с основами JS и HTML, материал и темы, которые обязательны для курса (возможно не увидел)
  • разработчики, которые приходят на курс, рассматривают данный курс, как доп. навыки, которые помогут работать на новых проектах не только с back-end средой, но и вести frontend разработку. При этом основной упор будет выполняться именно на бекенд. В связи с этим привязка к IDE, указанной в курсе, Visual Code, может быть не применима в большнистве проектов на практике и разработчикам.
  • Работа с Markdown и ReadMe так же может быть излишней, т.к. большинство разработчиков работает с техническим описанием в рамках своего проекта, кроме того, могут использоваться другие инструментарии (confluence). Не относится на прямую к текущему курсу разработки angular 10
  • За счет сокращения времени на развертывание среды, можно уделить больше внимания TypeScript-у, по крайней мере актуально для меня
  • требуется более жесткий тайминг
